## 🚀 Exercises Overview
Here is a quick summary of the concepts covered and implemented in each application:

* **Customer.java & myDate.java:** Customer profile and registration date representation (Classes, private variables, constructors).
* **CustomerList.java & CustomerApp.java:** Interactive menu application managing a 1D array of customer objects (Object arrays, data insertion, lookup algorithms).
* **VotingApp.java:** Election results analysis across multiple districts and candidates (2D arrays, percentages, conditional logic).
* **ShapeApp.java:** Array of random geometric shapes demonstrating late binding (Inheritance, Polymorphism).
* **Clock.java & ClockApp.java:** 24-hour digital clock simulation with real-time ticking (Encapsulation, state management, thread sleep).
* **AMPMClock.java & AMPMClockApp.java:** 12-hour clock functionality extending the base clock without code duplication (Inheritance, Overriding).
