Welcome to the first problem set for the **Computer Programming in Java** course (AUEB).

## 🚀 Exercises Overview
Here is a quick summary of the concepts covered and implemented in each application:

* **App1.java & App2.java:** Prime number validation (Demonstrating Instance vs. Static methods).
* **App3.java:** Word counting from raw text input (String manipulation & I/O).
* **App4.java:** Recursive calculation of the sum of a number's digits (Recursion).
* **App5.java:** Simple calculator simulation (Basic Object-Oriented Programming concepts).
* **App6.java:** Parking ticket change calculator (Algorithm to find the minimum number of banknotes/coins).
* **App7.java:** Electronics store product management (Classes, constructors, state management, and stock/sales logic).
